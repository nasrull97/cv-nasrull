Tentang Saya

Saya memulai perjalanan di dunia teknologi sejak duduk di bangku SMK dengan jurusan Rekayasa Perangkat Lunak, kemudian melanjutkan ke perguruan tinggi di bidang Teknik Informatika. Meski studi formal tidak saya selesaikan hingga akhir, semangat belajar dan eksplorasi tidak pernah berhenti.

Karier saya dimulai di industri konveksi, lalu berkembang di dunia garment sebagai Koordinator Proses Sample. Setelah itu, saya mendalami bidang pemisahan warna untuk printing sablon, di mana saya menguasai software desain profesional.
Seiring waktu, saya kembali ke akar minat saya di dunia pemrograman, khususnya di pengembangan frontend dan backend.

Saya adalah pribadi yang menyukai tantangan baru, cepat beradaptasi, dan memiliki keinginan besar untuk terus tumbuh baik dalam hal teknis maupun dalam kolaborasi lintas bidang.
